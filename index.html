ImpactJS click/hover example

This code show how to add click / mouse in / mouse out ( hover ) [with proper entity zIndex check ) functionality to your ImpactJS games, hope it will helps to you :)

Just remember to add this line into your init function in main.js:

ig.input.bind( ig.KEY.MOUSE1, 'mouse1' );
